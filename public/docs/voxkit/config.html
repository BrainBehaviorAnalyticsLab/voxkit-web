voxkit.config

This module provides access to configurable info about the application.

Configurations

  • App config: application metadata and provenance (app_config).
  • Pipeline config: pipeline steps and UI wiring (pipeline_config).
  • Startup config: launch-time constants and defaults (startup_config).
  • Logging config: rotating file logger setup (logging_config).

Only app_config and pipeline_config are dynamic post-build; they are loaded from YAML files under the active profile and can change without a rebuild. startup_config and logging_config are baked in at build time.

Profile System

Configs are organized into profiles under config/profiles//. The active profile is specified in config/profile.txt.

Use get_active_profile() to check which profile is active. Use get_profile_config_path() to get the path to the active profile's directory.

21
22    Returns the correct config path whether running from source or as a

24
25    Returns:
26        Path to the config directory
27    """
28    # Check if running as PyInstaller bundle

def resolve_config_file(filename: str) -> pathlib.Path: 
 71def resolve_config_file(filename: str) -> Path:
 72    """Resolve a config file path with fallback to default profile.
 73
 74    Looks for the file in the active profile first, then falls back to
 75    the default profile if not found. This allows profiles to only
 76    override the files they need to change.
 77
 78    Args:
 79        filename: The config file name (e.g., "app_info.yaml")
 80
 81    Returns:
 82        Path to the config file (from active profile or default)
 83
 84    Raises:
 85        FileNotFoundError: If file not found in active or default profile
 86    """
 87    config_root = get_config_root()
 88    profile = get_active_profile()
 89
 90    # Try active profile first
 91    active_path = config_root / "profiles" / profile / filename
 92    if active_path.exists():
 93        return active_path
 94
 95    # Fall back to default profile
 96    default_path = config_root / "profiles" / "default" / filename
 97    if default_path.exists():
 98        return default_path
 99
100    # Fall back to legacy location (config root)
101    legacy_path = config_root / filename
102    if legacy_path.exists():
103        return legacy_path
104
105    raise FileNotFoundError(
106        f"Config file '{filename}' not found in profile '{profile}', "
107        f"default profile, or config root"
108    )
