VoxKit API Documentation

This is example API documentation for VoxKit. In production, this would be generated automatically using pdoc from the Python source code.

Module: voxkit.analyzer

The analyzer module provides core functionality for speech analysis and processing.

Class: SpeechAnalyzer

Main class for performing speech analysis operations including phonetic alignment, acoustic feature extraction, and voice quality measurements.

__init__(sample_rate: int = 16000, language: str = 'en')

Initialize a new SpeechAnalyzer instance.

Parameters:

  • sample_rate (int) - Audio sample rate in Hz (default: 16000)
  • language (str) - Language code for phonetic models (default: 'en')

load_audio(file_path: str) → AudioData

Load an audio file for analysis.

Parameters:

  • file_path (str) - Path to audio file (WAV, MP3, FLAC, OGG)

Returns:

  • AudioData - Loaded audio data object

Raises:

  • FileNotFoundError - If audio file doesn't exist
  • ValueError - If file format is not supported

extract_features(audio: AudioData, features: list[str]) → dict

Extract acoustic features from audio data.

Parameters:

  • audio (AudioData) - Audio data to analyze
  • features (list[str]) - List of features to extract: 'pitch', 'formants', 'intensity', 'mfcc'

Returns:

  • dict - Dictionary mapping feature names to numpy arrays

Example:

analyzer = SpeechAnalyzer()
audio = analyzer.load_audio("speech.wav")
features = analyzer.extract_features(audio, ['pitch', 'formants'])
print(features['pitch'].shape)  # (num_frames,)

align_phonetic(audio: AudioData, transcript: str) → Alignment

Perform forced phonetic alignment using Montreal Forced Aligner.

Parameters:

  • audio (AudioData) - Audio data
  • transcript (str) - Text transcription of the audio

Returns:

  • Alignment - Phonetic alignment with word and phoneme boundaries

Class: AudioData

Container for audio data and metadata.

Attributes:

  • samples (np.ndarray) - Audio samples as numpy array
  • sample_rate (int) - Sample rate in Hz
  • duration (float) - Duration in seconds
  • channels (int) - Number of audio channels

resample(target_rate: int) → AudioData

Resample audio to a different sample rate.

normalize() → AudioData

Normalize audio amplitude to [-1, 1] range.

Getting Started

Here's a complete example of using VoxKit for speech analysis:

from voxkit.analyzer import SpeechAnalyzer
from voxkit.export import export_to_csv, export_textgrid

# Initialize analyzer
analyzer = SpeechAnalyzer(sample_rate=16000, language='en')

# Load audio file
audio = analyzer.load_audio("interview.wav")

# Extract acoustic features
features = analyzer.extract_features(audio, ['pitch', 'formants', 'intensity'])

# Perform phonetic alignment
transcript = "Hello world, this is a test recording"
alignment = analyzer.align_phonetic(audio, transcript)

# Export results
export_to_csv(features, "features.csv")
export_textgrid(alignment, "alignment.TextGrid")
